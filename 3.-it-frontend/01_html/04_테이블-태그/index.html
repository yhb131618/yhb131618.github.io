04 테이블 태그

#html

1. 테이블을 만들어주는 태그 #

1. table 태그 #

  1. 표의 자리를 지정해주는 태그
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>표를 만들어주는 table, tr, th, td</title>
    <style>
        table, th, td {
            border: 1px solid black;
            border-collapse: collapse;
        }
    </style>
</head>
<body>
    <!--table 태그: 표를 만들어주는 태그-->
    <table>
        <!--tr 태그: 표에 행을 추가-->
        <tr>
            <!--th 태그: 제목 셀-->
            <th>Java</th>
            <!--td 태그: 내용 셀-->
            <td>변수</td>
            <td>클래스</td>
        </tr>
        <tr>
            <th>Oracle</th>
            <td>DML</td>
            <td>DDL</td>
        </tr>
    </table>

</body>

2. tr태그 #

  1. 표에 행을 추가해주는 태그

3. th태그 #

  1. 행의 제목이 되는 태그

4. td태그 #

  1. 행의 내용이 되는 태그
  2. th, td 태그는 tr 태그의 자식 태그로 선언한다.

5. 행 또는 열을 병합하는 colspan, rowspan 속성 #

  1. colspan : 지정한 셀의 개수만큼 가로로 병합

  2. rowspan : 지정한 셀의 개수만큼 세로로 병합

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>셀을 병합해주는 colspan, rowspan</title>
  <style>
      table, th, td {
          border: 1px solid black;
          border-collapse: collapse;
      }

      tr {
          height: 30px;
      }

      th, td {
          width: 200px;
          height: 30px;
      }
  </style>
</head>
<body>
  <table>
      <tr>
          <!--rowspan 속성: 세로 셀을 병합-->
          <th rowspan="2">이름</th>
          <td rowspan="2">고기천</td>
          <th rowspan="2">연락처</th>
          <td rowspan="2">011-1111-1111</td>
      </tr>
      <tr>
          <th>주소</th>
          <td colspan="3">서울시 서초구</td>

      </tr>
      <tr>
          <th>인사말</th>
          <td colspan="3">안녕하세요.</td>
      </tr>
      <tr>
          <th></th>
          <td></td>
      </tr>
  </table>
</body>
</html>

6. 테이블에 제목을 붙여주는 caption #

  1. table 태그 바로 뒤에 사용. 표 위쪽 중앙에 제목이 표시된다. caption 태그안에 다른 태그를 사용하여 스타일을 입힐 수도 있다.
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>표에 제목을 붙여주는 caption</title>
  <style>
      table, th, td{
          border: 1px solid black;
          border-collapse: collapse;
      }
  </style>
</head>
<body>
  <table>
      <caption>
          <strong style="color: red">Frontent Framework</strong>
          <p>요즘 자주 사용되는 SPA Framework</p>
      </caption>
      <tr>
          <th>언어</th>
          <th>개발회사</th>
          <th>배포년도</th>
      </tr>
      <tr>
          <th>React</th>
          <td>메타(구 페이스북)</td>
          <td>2013</td>
      </tr>
      <tr>
          <th>Vue</th>
          <td>MIT</td>
          <td>2012</td>
      </tr>
  </table>
</body>
</html>

7. 여러 열을 묶어서 스타일을 지정하는 colgroup, col #

  1. colgroup 태그는 컬럼들의 그룹을 생성. caption 뒤 tr, td 전에 사용.

  2. col 태그는 colgroup 태그 내에서 컬럼에 적용할 스타일을 지정할 수 있는 태그

  3. col 태그의 개수는 테이블의 열의 개수와 동일해야 된다.

  4. 행을 그룹화하는 태그는 존재하지 않는다.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>여러 열을 묶어서 스타일을 지정하는 colgroup, col</title>
  <style>
      table, th, td{
          border: 1px solid black;
          border-collapse: collapse;
      }
  </style>
</head>
<body>
  <table>
      <caption>
          <strong style="color: red">Frontent Framework</strong>
          <p>요즘 자주 사용되는 SPA Framework</p>
      </caption>
      <!--colgroup 태그: caption태그 뒤, tr태그 전에 선언
          열을 묶어주는 역할-->
      <colgroup>
          <!--col 태그: colgroup 내부에 선언 가능한 태그
          열마다 스타일을 다르게 줄 수있는 태그
          col 태그의 개수는 최대 테이블에 존재하는 컬럼의 개수-->
          <col style="background-color: aqua;">
          <col style="background-color: yellow;">
          <col style="background-color: lightgreen;">
      </colgroup>
      <tr>
          <th>언어</th>
          <th>개발회사</th>
          <th>배포년도</th>
      </tr>
      <tr>
          <th>React</th>
          <td>메타(구 페이스북)</td>
          <td>2013</td>
      </tr>
      <tr>
          <th>Vue</th>
          <td>MIT</td>
          <td>2012</td>
      </tr>
  </table>
</body>
</html>

8. 표에 설명을 붙여주는 figure, figcaption #

  1. table 태그를 figure 태그로 감싼 후 table 태그 전에 figcaption 태그를 사용해서 table에 대한 설명을 작성할 수 있다.

9. 표의 구조를 잡아주는 thead, tbody, tfoot #

  1. thead 태그는 테이블의 제목 행

  2. tbody 태그는 테이블의 본문 부분

  3. tfoot 태그는 테이블의 요약 부분

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>표의 구조를 잡아주는 thead, tbody, tfoot</title>
  <style>
      table, th, td {
          border: 1px solid black;
          border-collapse: collapse;
      }

      thead {
          background-color: yellow;
      }

      tfoot {
          background-color: lightgreen;
      }
  </style>
</head>
<body>
  <table>
      <caption>데브옵스 과정 현황</caption>
      <thead>
          <tr>
              <th>구분</th>
              <th>신청인원</th>
              <th>재적인원</th>
              <th>현재인원</th>
          </tr>
      </thead>
      <tbody>
          <tr>
              <th>1기</th>
              <td>30</td>
              <td>5</td>
              <td>25</td>
          </tr>
          <tr>
              <th>2기</th>
              <td>30</td>
              <td>2</td>
              <td>28</td>
          </tr>
          <tr>
              <th>3기</th>
              <td>28</td>
              <td>4</td>
              <td>24</td>
          </tr>
          <tr>
              <th>4기</th>
              <td>17</td>
              <td>1</td>
              <td>16</td>
          </tr>
          <tr>
              <th>5기</th>
              <td>26</td>
              <td>2</td>
              <td>24</td>
          </tr>
      </tbody>
      <tfoot>
          <tr>
              <th>합계</th>
              <td>131</td>
              <td>14</td>
              <td>117</td>
          </tr>
      </tfoot>
  </table>
</body>
</html>
